# DeepMindz LMS

A complete Learning Management System (LMS) developed using PHP, MySQL, HTML, CSS, and Object-Oriented Programming (OOP) concepts during my internship at DeepMindz.

This project provides separate dashboards for Admins and Students, similar to a university learning portal.

---

# 🚀 Features

## 👨‍💼 Admin Dashboard
The admin panel is built using PHP OOP concepts and allows administrators to:

- Add and manage students
- Add and manage users
- Create and manage courses
- Add modules under specific courses
- Organize course content efficiently
- Manage student access

---

## 🎓 Student Dashboard
Students can:

- Register and create an account
- Login securely using credentials
- Access enrolled courses
- View course modules
- Navigate learning content in a university-style dashboard
